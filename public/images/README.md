# Images Directory

This directory should contain the following placeholder images:

- `profile.jpg` - Your profile picture (recommended: 800x800px, square)
- `project-ecommerce.jpg` - Screenshot/image for E-Commerce Platform project
- `project-taskmanager.jpg` - Screenshot/image for Task Management App project
- `project-social.jpg` - Screenshot/image for Social Media Dashboard project
- `og-image.jpg` - Open Graph image for social media sharing (1200x630px)

## Image Guidelines

- Use `.jpg` or `.png` format
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Keep file sizes reasonable (< 500KB per image)
- Use descriptive alt text in components for accessibility

## Adding Your Images

Replace these placeholder files with your actual images. The components will automatically use the images once you add them to this directory.
